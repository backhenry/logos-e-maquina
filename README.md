# O Logos e a Máquina

> *Como a Linguagem Molda a Realidade e Por Que Dominar a Engenharia de Prompt é Dominar o Futuro*

[![License: CC BY-SA 4.0](https://img.shields.io/badge/License-CC%20BY--SA%204.0-lightgrey.svg)](https://creativecommons.org/licenses/by-sa/4.0/)
[![Made with VitePress](https://img.shields.io/badge/Made%20with-VitePress-42b883)](https://vitepress.dev/)

Um livro sobre engenharia de prompt escrito do lugar mais inesperado: a filosofia da linguagem. Do grito do bebê em Lacan aos Transformers do Google. De Confúcio a Helen Keller. Dos russos que veem dois azuis aos engenheiros que perdem sondas da NASA por unidades de medida ambíguas.

A tese central é simples e desconfortável: **engenharia de prompt não é uma habilidade técnica nova. É a redescoberta de uma habilidade humana muito antiga — falar com precisão.** O que mudou foi o interlocutor.

## Por que este livro existe

Há duas literaturas sobre prompt engineering circulando hoje. Uma é técnica e seca — listas de frameworks, exemplos triviais, promessas de produtividade. A outra é mística e vaga — "converse com a IA como se ela fosse seu amigo". Nenhuma das duas serve a quem precisa, de fato, conseguir resultados consistentes de Grandes Modelos de Linguagem em contextos profissionais sérios.

Este livro é uma terceira via: tratar prompt engineering como o que ele realmente é — **uma disciplina linguística aplicada**, com 2.500 anos de história filosófica por trás e arquitetura matemática rigorosa por baixo. Quem entende as duas coisas escreve prompts melhores. Quem entende uma só pisa em buracos previsíveis.

## Como ler

Você pode ler do começo ao fim, em ordem, como uma narrativa que vai construindo argumento. Recomendado para quem está começando.

Você pode ler por partes. Cada uma das seis partes funciona como um livro pequeno autocontido. Se você é um pragmático com pressa, vá direto para a **Parte IV** (frameworks) e a **Parte V** (aplicação). Se você é um curioso filosófico, comece pela **Parte I** e vá pelo caminho longo.

Você pode usar como referência. O **Glossário** no fim cobre todos os termos técnicos e filosóficos.
